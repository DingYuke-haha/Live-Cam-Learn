## ðŸš€ How to Run (Development)

This project is built with **Flutter** and can be opened and run using **Android Studio**. To run the app on an Android device:

1. **Clone the repository**

   ```bash
   git clone https://github.com/DingYuke-haha/Live-Cam-Learn.git
   cd Live-Cam-Learn
   ```

2. **Open in Android Studio**

   * Launch Android Studio.
   * Choose **Open an existing project**.
   * Select the project directory.

3. **Install dependencies**
   Android Studio / Flutter should automatically fetch required packages.
   If needed, run:

   ```bash
   flutter pub get
   ```

4. **Connect an Android device**

   * Plug in your Android phone via USB.
   * Enable **USB debugging** on the device (Developer Options).

5. **Run the app**

   * Select your connected device in Android Studio.
   * Click **Run** â–¶ï¸ to build and install the app.
